🔗 เชื่อมต่อ Google Sheets

1
เปิด Google Sheets ใหม่ (sheets.new) แล้วตั้งชื่อตามต้องการ
2
คลิก Extensions → Apps Script แล้ววาง code จากไฟล์ Code.gs ที่ได้รับ
3
คลิก Deploy → New deployment
— Type: Web app
— Execute as: Me
— Who has access: Anyone
กด Deploy แล้วอนุญาต permissions
4
คัดลอก Web app URL แล้วนำมาวางด้านล่าง
URL จะขึ้นต้นด้วย https://script.google.com/macros/s/
